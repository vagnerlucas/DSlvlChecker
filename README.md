DSlvlChecker
============

- Repositório criado em 24/04/2013

- DarkSouls PC PvP/PvE Level Checker

Projeto que consiste em "ler" os atributos de jogadores co-op ou pvp do game Dark Souls Prepare To Die Edition (PC).
A finalidade desse projeto é criar uma ferramente capaz de detectar irregularidades nos status de jogadores aliados ou inimigos, que estejam com alterações em seus personagens. Isso facilita um jogo saudável e sem trapaças.

- Linguagem de desenvolvimento

Este aplicativo foi desenvolvido em linguagem Object Pascal (Delphi). IDE de utilização: Delphi XE3.
